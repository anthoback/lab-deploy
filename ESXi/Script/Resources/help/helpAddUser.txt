AddUser.py - General help

Description : Allows you to add Users to a LAB (Guaccamole) based on the file Template_Users.xlsx

Usage : You can run the script with or without arguments 

With :
    Example : python3 AddUser Lab1 
                               1    
 - arg 1 = input the Name of the LAB 

Without :
    Example of sequence of command to add Users :
     Please enter the name of the lab >> Lab1


NB: You can choose or not the password By putting it in the 2nd column 
otherwise the password will be generated automatically and indicated in the corresponding file in the Pwd folder .
